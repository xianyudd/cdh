## 🚀 v0.1.1

发布日期：2025-11-07

### 🧩 更新内容

* ✨ **新增：** `install.sh` 安装脚本，支持 Linux / macOS / fish / zsh / bash 的一键安装与 shell 集成。
* 🧭 **改进：** TUI（基于 stderr 输出界面，stdout 仅输出目录路径），更好兼容 shell 自动 cd。
* ⚙️ **优化：** Frecency 算法调优，使常用目录排序更智能；增强 Controller 模块稳定性。
* 📦 **完善：** 新增 LICENSE、README、CI workflow、打包与自动发布逻辑。

### 🧰 构建支持

* ✅ Linux: x86_64-unknown-linux-gnu
* ✅ macOS: x86_64-apple-darwin / aarch64-apple-darwin

### 💡 安装方式

```bash
curl -fsSL https://raw.githubusercontent.com/xianyudd/cdh/main/scripts/install.sh | bash
```

> 💬 提示：TUI 界面走 `stderr`，stdout 仅输出目录字符串，可直接配合 shell alias 使用。
> 📂 安装后执行 `cdf` 可快速进入交互界面，选定后自动 cd 到目标目录。
