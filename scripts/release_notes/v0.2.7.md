## 🚀 v0.2.7

发布日期：2026-06-07

### 🧩 更新内容

* 📦 **改进：** GitHub release tarball 现在包含完整 shell installer payload。
* 🧰 **新增：** 支持从 release 包内运行 `install.sh`，无需再访问 `raw.githubusercontent.com` 下载子安装器。
* 📝 **更新：** README 增加 release tarball 安装方式，适合 raw GitHub 不稳定的 VM 环境。

### 🧰 构建支持

* ✅ Linux: x86_64-unknown-linux-gnu
* ✅ macOS: x86_64-apple-darwin / aarch64-apple-darwin

### 💡 安装方式

```bash
curl -fsSL https://github.com/xianyudd/cdh/releases/download/v0.2.7/cdh-v0.2.7-x86_64-unknown-linux-gnu.tar.gz \
  | tar -xz
cd cdh-v0.2.7-x86_64-unknown-linux-gnu
bash --noprofile --norc install.sh
```
